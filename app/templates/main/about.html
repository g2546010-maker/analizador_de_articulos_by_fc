Acerca del Sistema

Sistema de Gestión de Artículos Académicos (SGAA)

Sistema web diseñado para facilitar la gestión de producciones académicas de un Cuerpo Académico, reduciendo significativamente la captura manual de datos mediante extracción automática desde PDFs.

Tecnologías Utilizadas
  • Flask 3.0 - Framework web
  • SQLAlchemy - ORM para base de datos
  • SQLite - Base de datos (desarrollo)
  • PyPDF2 / pdfplumber - Extracción de PDFs
  • openpyxl - Generación de Excel
  • Bootstrap 5 - Interfaz de usuario
  • Python threading - Tareas en background
Funcionalidades Principales
Gestión de Artículos
  • CRUD completo
  • Validaciones
  • Paginación
Procesamiento PDF
  • Extracción automática
  • Pre-llenado de formularios
  • Validación de archivos
  • Almacenamiento seguro
Catálogos
  • Tipos de producción
  • Estados
  • LGAC
  • Indexaciones
Arquitectura

El sistema implementa el patrón MVC (Modelo-Vista-Controlador) organizado por módulos:

  • Modelos - Entidades de base de datos (Artículo, Autor, Revista, etc.)
  • Vistas - Blueprints de Flask para rutas HTTP
  • Controladores - Lógica de negocio
  • Servicios - Funcionalidades especializadas (PDF, Excel, Background)
Versión Actual

MVP v1.0 - Producto Mínimo Viable con funcionalidades esenciales.
Fecha de desarrollo: Diciembre 2025
